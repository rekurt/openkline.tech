Monospace tabular metric readout with bull/bear-colored delta — for tickers, stat rows, marketing proof numbers.

```jsx
<PriceStat label="BTC/USDT" value="67,412.50" delta={2.31} />
<PriceStat label="Unit tests" value="440+" size="lg" />
```

`delta` as a number auto-formats to `▲ 2.31%` / `▼ 0.84%` in bull/bear color.
